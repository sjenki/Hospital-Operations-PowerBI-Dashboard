# Hospital Operations Power BI Dashboard
This portfolio project showcases an interactive Power BI dashboard designed to analyze hospital operations, patient demographics, financial performance, and clinical outcomes. The dashboard demonstrates the use of data visualization, DAX measures, and executive reporting techniques to transform healthcare data into meaningful insights for operational and strategic decision-making.

# Project Objectives
* Develop a multi-page healthcare analytics dashboard using Power BI.
* Create executive-level KPIs to monitor hospital performance.
* Visualize patient demographics and clinical outcomes.
* Demonstrate proficiency in DAX, data modeling, and interactive reporting.
* Present healthcare data in a format that supports informed decision-making.

---

## Dashboard Pages

### 📊 Executive Summary

Provides a high-level overview of hospital performance through key performance indicators (KPIs), including:

* Total Admissions
* Total Billing
* Average Billing per Patient
* Average Length of Stay (LOS)
* Admissions by Month
* Billing by Diagnosis
* Billing by Insurance Provider
* Admissions by Type

---

### 👥 Patient Demographics

Analyzes patient population characteristics, including:

* Age Distribution
* Gender Distribution
* Diagnosis by Age Group
* Insurance Provider by Age Group
* Blood Type Distribution
* Most Common Diagnosis

---

### 🏥 Hospital Performance

Evaluates operational performance across hospitals using:

* Admissions by Hospital
* Billing by Hospital
* Average Billing
* Average Length of Stay
* Normal Test Results %
* Test Results by Category
* Admissions by Type

---

### 🩺 Clinical Outcomes

Examines patient outcomes and quality metrics, including:

* Normal vs. Abnormal Test Results
* Medication Usage
* Billing by Diagnosis
* Test Results by Diagnosis
* Most Common Medication
* Average Length of Stay
